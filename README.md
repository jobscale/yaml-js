# @jobscale/yaml-js

YAML JSON Convertor

## Installation
```
npm i -g @jobscale/yaml-js
```

## Examples

### YAML to JSON
```
yaml2json test/sample.yaml
```

### JSON to YAML
```
json2yaml test/sample.json
```

# js-yaml

## Examples

```
# show version
js-yaml -v
js-yaml --version

# show help
js-yaml -h
js-yaml --help

# yaml to json
js-yaml test/sample.yaml

# json to yaml
js-yaml test/sample.json

# yaml to json
js-yaml < test/sample.yaml

# json to yaml
js-yaml < test/sample.json
```
